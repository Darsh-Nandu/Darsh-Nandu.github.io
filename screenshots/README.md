# Screenshots

Add screenshots of your portfolio here after deploying:

- `hero.png`     — screenshot of the hero/home section
- `projects.png` — screenshot of the projects/changelog section
- `about.png`    — screenshot of the about + stack sections
- `contact.png`  — screenshot of the contact terminal section

**Tip:** Use the browser's built-in screenshot tool:
- Chrome: DevTools → Ctrl+Shift+P → "Capture full size screenshot"
- Firefox: Right-click → "Take Screenshot"

These are referenced in the README.md for the GitHub preview.
